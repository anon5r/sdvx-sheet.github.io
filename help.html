• 注意事項
    • 播放器由 Javascript 和 html5 所寫的，並且還需要 Web Audio API 的支持，因此瀏覽器若不支援則無法使用
    • 推薦使用 Chrome 瀏覽器，Firefox 沒有測試太多但目前沒太大問題
  • 第一排
    • 選擇歌曲，並按下 Load，旁邊會顯示出歌曲讀取進度，等到 100% 之後就可以按下 Play 播放
    • 歌曲播放中，可以按下 Stop 來暫停播放，要繼續播放則可以再按一次 Play
    • 可以使用空白鍵來當作 Stop 和 Play 的功能
    • Copy 可以將目前的歌曲、速度以及時間記錄在網址列中，只要用這個網址列就可以直接選到一模一樣的狀態
  • 第二排
    • BPM 為顯示目前歌曲現在時間的 BPM
    • Speed 為譜面速度，可自由調整，調整完之後請記得重新按 Load
    • BPM × Speed 顯示兩者相乘之後的譜面速度
    • Time 為目前歌曲的時間，不推薦手動調整，可以使用第三排的 bar 來調整歌曲時間
    • Volume 可以調整歌曲音量
  • 第三排
    • 左邊的 bar 為目前歌曲的時間位置，可以拖曳來調整時間
    • 調整時間也可以使用滑鼠滾輪
    • Shift 為當音樂和譜面不對拍的時候，可以自由調整音樂的 timing，例如音樂太慢想要往後延遲一點就設定一個 > 0 的數字，單位為秒
  • 第四排
    • Music Speed 為調整整個歌曲的速度，例如調為 0.7 整個歌曲會放慢（音調也會降低），可以拿來當慢速練習用
    • Contant BPM 打勾之後，當遇到變速的時候，Speed 會自動調整讓 BPM × Speed 的值不會改變（會使得譜面在變速的時候稍微停頓一下），用來對付一些可怕的變速歌 ex. Lunartic Rough Party
    • Blur / Red Color 可以自由調整藍色跟紅色旋鈕的顏色
